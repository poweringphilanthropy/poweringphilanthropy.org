An Age Gate addon to check the age of users registering or using WooCommerce

== Description ==
This addon for Age Gate allows you to Age Gate users when registering for your site. It can add the age check for the standard Wordpress registration and WooCommerce registration and checkouts.

__Features__

* Inherits your preferences from the main Age Gate plugin
* Gives option to store the users date of birth when registering
* Allows you to set a different required age for registering
* Support for WordPress and WooCommerce registrations

== Installation ==
1. Upload the 'age-gate-user-registration' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit 'Age Gate' admin section and adjust your configuration in the addons tab.

__Important__
This is an early release so please report any issues so they can be resolved quickly

== Frequently Asked Questions ==
